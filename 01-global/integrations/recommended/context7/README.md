# Context7 integration

Context7 là integration `khuyến nghị` cho docs library/framework mới nhất.

- Auto-check khi cài runtime
- Thiếu thì auto-install
- Verify bằng cách gọi `context7-mcp --help`

Nó bổ trợ cho `codebase-memory-mcp`:

- `codebase-memory-mcp` hiểu codebase hiện tại
- `context7` hiểu docs/upstream hiện tại
