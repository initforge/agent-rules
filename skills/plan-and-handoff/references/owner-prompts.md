## Optional prompts

### Plan

```text
Plan this work as an executable contract. Scale detail to the task; ask only a meaningful question. Include outcome, scope, interfaces/files, proof, and next safe action. For long/resumable work, add disjoint slices and a detailed source/injection ledger.
```

### Execute

```text
Execute the approved plan automatically. Main agent owns intent, integration, final review, and final status. Delegate only disjoint slices with context capsules; continue dependency-ready work without asking me to relay phases. Match proof to each claim and report genuine blockers only.
```
