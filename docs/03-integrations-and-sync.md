# Integrations and sync

Code intelligence uses one pinned Codebase Memory MCP binary with separate Codex, Grok and Antigravity config adapters. The retired graph tool must not appear in active source or runtime configuration.

Caveman is optional and owned by its upstream repository. This repo stores lifecycle wrappers only; it does not vendor upstream skills or scripts.

Mirrors flow only canonical → build → runtime. Runtime import requires an explicit reviewed diff; timestamp-based two-way merging is forbidden.
