# Harness glossary

| Term | Meaning |
|---|---|
| **PAF** | Plan Artifact Format — locked multi-phase plan. |
| **HB-1** | Plan mode does not edit source before an explicit pivot. |
| **HB-2** | Execution begins only after a clear implementation pivot. |
| **HB-3** | A pasted plan is not an execution scope lock by itself. |
| **HB-4** | Two or more files cannot use the tiny execution lane. |
| **HB-5** | Locked execution closes the complete in-scope slice. |
| **SGP** | Slice Gate Protocol for acceptance-criteria and ledger checks. |
| **L0/L1/L2** | Capability tiers; use the weakest suitable executor first. |
